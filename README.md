  > 2.Bln control

  > Set the total number of LED strip:Total
  >

  ![](images/water.png)

- Set bln control

  > Set the total number of LED strip:Total
  >

  ![](images/bln.png)

  > Set the total number of LED strip:Total
  >


  > Set the time of blink:Time
  >

  ![](images/blink.png)

  > Set the total number of LED strip:Total
  >


  > Set the start number which :Start
